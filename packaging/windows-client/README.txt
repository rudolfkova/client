DnD — клиент для Windows (x64)
================================

Содержимое этой папки можно целиком скопировать на ПК с Windows и запускать отсюда.

Запуск игры
-----------
  • Двойной щелчок по run-game.bat — откроется консоль, в ней нужно ввести:
      1) адрес gateway (host:port), Enter = localhost:8080
      2) email и пароль
  • Либо вручную: открыть cmd.exe, перейти в эту папку (cd), выполнить game.exe

Запуск редактора мира
----------------------
  • run-editor.bat — далее логин и пароль в консоли.

Файлы
------
  game.exe, editor.exe — собранные клиенты (ассеты тайлов и каталог уже вшиты в exe).
  Папка data/ — копия из репозитория (справочно; для обычной игры не обязательна рядом с exe).

Требования: Windows 10/11 x64, доступен ваш game-gateway по сети.
